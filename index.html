Student Data Merge Tool

Developed for Concord Consortium by Saul Amster

Step 1:


Click the button and select a folder that contains only .xlsx files in it. These files should have a column labeled "Teachers". Also, the first column should be the Student ID column.

Step 2:


Click the button to read the files into the program. This could take several minutes.

Step 3:


Enter the names of the teachers whose student data you wish to extract separated by commas in the text area below. These names must match the names found in the data imported during Step 2. Click the button to enter the teacher names and verify that they exist in the data. If this field is left empty, all data will be merged.

Step 4:


Click the button to let the program filter all the data and extract the desired student data.

Step 5:


Click the button to select the directory to export the filtered data to as a new .xlsx file.
